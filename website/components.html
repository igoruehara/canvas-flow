Nodes básicos do fluxo

Mensagem

Envia texto para o usuário e pode usar templates com `context.slots`.

Mensagem rica

Renderiza botões, quick replies, listas, carrosséis, imagem, documento e fluxos interativos de atendimento.

Input

Coleta resposta do usuário, salva em `responseName` e valida por tipo, regex ou LLM.

API

Executa chamada HTTP direta dentro do fluxo, com headers, query, body e mapeamento de resposta.

Condição

Decide o próximo caminho por JavaScript ou LLM a partir do contexto atual.

Fim

Finaliza a execução atual e marca a conversa como encerrada quando o fluxo pede.

Grupo

Organiza visualmente blocos do canvas sem alterar a lógica de execução.

IA e orquestração

Agente

Node LLM principal com modelo, temperatura, prompt, Agent OS, skills, subagents, rules e guardrails.

Azure OpenAI

Variação especializada para deployment Azure OpenAI em fluxos que precisam fixar esse provedor no node.

Agent Plan

Cria um contrato visual de planejamento em `context.slots.agentPlan` para orientar ou fixar a estratégia antes das tools.

Aprovação humana

Pausa a execução antes de ações sensíveis e continua quando a API ou operador envia aprovar/reprovar.

RAG, arquivos e dados

RAG IA Gen

Busca documentos, aplica filtros e usa LLM para gerar a resposta final com contexto recuperado.

Milvus

Executa busca vetorial ou híbrida no Milvus/Zilliz e salva documentos encontrados no contexto.

Azure AI Search

Consulta índices do Azure AI Search e alimenta o fluxo com documentos relevantes.

Azure Blob Storage

Grava e lê payloads, documentos ou artefatos em containers Azure Blob.

Arquivos

Lê upload ou URL, extrai texto, responde com LLM ou gera/edita documentos como TXT, MD, CSV, JSON, HTML, DOCX, XLSX e PDF.

MongoDB

Executa insert, find, update, upsert, delete, count e aggregate em collections de negócio.

Contexto

Cria ou atualiza slots por JSON dinâmico, JavaScript ou LLM para preparar dados dos próximos nodes.

Dashboard

Monta cards, tabelas e gráficos a partir de trace, MongoDB, API ou Milvus, com apoio opcional de LLM.

Ações e integrações

MCP

Chama APIs como tools internas ou conecta servidores MCP externos por Streamable HTTP, SSE ou WebSocket.

Webhook

Recebe eventos externos, envia callbacks ou escuta eventos assíncronos com autenticação por bearer, header ou query.

Debug

Registra snapshot do contexto atual para entender slots, input, flow ativo e decisões durante o teste.

Controle e automação

Loop

Repete uma saída até atingir limite de iterações ou até uma condição JavaScript interromper a repetição.

Roteador de fluxo

Envia a conversa para outro flow por regras JavaScript ou decisão LLM, com fallback configurável.

Cron

Agenda execuções por intervalo, dia, semana ou mês e injeta texto/slots iniciais na rotina automática.
